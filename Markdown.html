What is it used for

Markdown is a lightweight markup language commonly used to format and style plain text documents. It is often employed to create documentation, README files, and other types of text-based content. Markdown is designed to be easy to read and write, and it can be converted into HTML or other formats for display on the web or other applications.

 

Markdown will be perfect for you, especially if you publish regularly on the Internet, where HTML is prevalent: WordPress, Squarespace, Jekyll... Services like Trello or forums like Stack Overflow also support this language, and over time, you will find even more places that use it.
